Eco-Scroll.js

Eco-friendly jQuery scroll plugin with infinite possibilities

Github

What does it do?

Eco-Friendly

Elements will get removed when they go out of sight.

Infinite Scroll

It can scroll with NO Boundary.

Possibilities

It can transform into many different tools

Why?

I wanted to create a infinite scroll plugin for my own use.

Demo

Behind the scenes

As you drag divs are appended and when dragging is done, divs outside of the viewport will be removed.
